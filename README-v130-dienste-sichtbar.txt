ULMIPOINT v130 – Dienste sichtbar in Organisation verwalten

Änderung gegenüber v129:
- organisation-admin.html zeigt nun einen eigenen Bereich „Dienste“.
- Dort gibt es Links zu:
  - dienstkatalog.html = Dienste zentral erstellen/bearbeiten
  - gruppen-dienste.html = Dienste pro Gruppe auswählen
- Bei jeder Gruppe/Bereich-Zeile gibt es zusätzlich den Button „Dienste“.
  Dieser öffnet direkt die Auswahl für genau diese Gruppe.

Logik bleibt unverändert:
Organisation erstellt Dienste.
Gruppe wählt Dienste und setzt Arbeit / Gruppe / Auto / Manuell.
Andere Gruppen werden nicht verändert.
